# s21_matrix — Библиотека для работы с матрицами на C

## Обзор проекта

Это учебный проект от **School 21**, представляющий собой реализацию статической библиотеки `s21_matrix.a` для обработки числовых матриц на языке **C11**. Библиотека предоставляет полный набор операций для работы с матрицами: создание/удаление, сравнение, арифметические операции, транспонирование, вычисление определителя, матрицы алгебраических дополнений и обратной матрицы.

### Технологии и стек

- **Язык:** C (стандарт C11, POSIX.1-2017)
- **Компилятор:** gcc
- **Стиль кода:** Google C++ Style Guide
- **Тестирование:** Check (unit-тесты)
- **Покрытие кода:** gcov / lcov / genhtml
- **Статический анализ:** cppcheck
- **Форматирование:** clang-format (Google style)
- **Профилирование памяти:** Valgrind

## API библиотеки

### Основные функции

| Функция | Описание |
|---------|----------|
| `s21_create_matrix(rows, columns, &result)` | Создание матрицы |
| `s21_remove_matrix(&A)` | Освобождение памяти матрицы |
| `s21_eq_matrix(&A, &B)` | Сравнение матриц (до 6 знаков после запятой) |
| `s21_sum_matrix(&A, &B, &result)` | Сложение матриц |
| `s21_sub_matrix(&A, &B, &result)` | Вычитание матриц |
| `s21_mult_number(&A, number, &result)` | Умножение матрицы на скаляр |
| `s21_mult_matrix(&A, &B, &result)` | Умножение двух матриц |
| `s21_transpose(&A, &result)` | Транспонирование матрицы |
| `s21_calc_complements(&A, &result)` | Матрица алгебраических дополнений |
| `s21_determinant(&A, &result)` | Вычисление определителя |
| `s21_inverse_matrix(&A, &result)` | Обратная матрица |

### Вспомогательные функции

| Функция | Описание |
|---------|----------|
| `is_matrix_valid(&A)` | Проверка корректности матрицы |
| `print_matrix(A)` | Вывод матрицы в консоль |
| `fill_matrix(&A, value)` | Заполнение матрицы значением |
| `sum_helper(&A, &B, &result, signer)` | Helper для сложения/вычитания |
| `calc_minor(&A, &temp, ex_row, ex_col)` | Вычисление минора |

### Коды возврата

| Код | Значение |
|-----|----------|
| `0` (`OK`) | Успех |
| `1` (`INCORRECT_MATRIX`) | Некорректная матрица (NULL, нулевые размеры, ошибка выделения памяти) |
| `2` (`CALC_ERROR`) | Ошибка вычисления (несовпадающие размеры, определитель невозможен) |

### Константы сравнения

| Константа | Значение |
|-----------|----------|
| `SUCCESS` (1) | Матрицы равны |
| `FAILURE` (0) | Матрицы не равны |

## Соглашения по разработке

### Стиль кода
- **Google C++ Style Guide** (адаптирован для C)
- Префикс `s21_` перед всеми публичными функциями
- Стандарт C11, POSIX.1-2017
- Запрет на устаревшие/legacy функции

### Тестирование
- Unit-тесты на базе библиотеки **Check**
- Минимальное покрытие: **80%** каждой функции
- Точность сравнения дробных чисел: **до 6 знаков после запятой** (`1e-6`)
- Тесты находятся в `tests/test.c`

### Математическая точность
- Сравнение матриц использует эпсилон `0.0000001` (1e-7)
- Определитель при поиске обратной матрицы: порог `1e-12`
